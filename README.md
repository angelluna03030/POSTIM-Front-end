
# POSTIM - Frontend

Una aplicación moderna construida con **Vite** y **React** para gestionar y compartir contenido.

## 🚀 Características

- Interfaz de usuario rápida y responsiva
- Construcción optimizada con Vite
- Componentes React reutilizables
- Enrutamiento dinámico
- Gestión de estado moderna

## 📋 Requisitos Previos

- Node.js (v16 o superior)
- npm o yarn

## 🛠️ Instalación

```bash
# Clonar el repositorio
git clone https://github.com/angelluna03030/POSTIM-Front-end.git

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

## 📦 Scripts Disponibles

- `npm run dev` - Inicia el servidor de desarrollo
- `npm run build` - Construye la aplicación para producción
- `npm run preview` - Vista previa de la build de producción

## 🏗️ Estructura del Proyecto

```
src/
├── components/    # Componentes reutilizables
├── pages/        # Páginas principales
├── styles/       # Estilos globales
└── App.jsx       # Componente raíz
```
